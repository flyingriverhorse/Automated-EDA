# Contributor Covenant Code of Conduct

This project follows the Contributor Covenant v2.1. By participating, you agree to abide by this code of conduct.

Summary:
- Be respectful and considerate in all interactions.
- No harassment, discrimination, or abusive language.
- Report unacceptable behavior to the project maintainers.

For the full text see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
